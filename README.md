# Online Examination System (Java)

A console-based Online Examination System developed using Core Java.  
The project simulates an online test environment with login authentication, MCQ-based exams, timer-based auto submission, and score evaluation.

## Features
- User Login Authentication  
- Update Profile Password  
- Multiple Choice Questions (MCQs)  
- Exam Timer with Auto Submit  
- Automatic Score Calculation  
- Logout and Session Termination  

## Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- Command Line Interface (CLI)
